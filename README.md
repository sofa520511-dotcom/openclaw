# 台灣運彩盤口分析工具

這是一個 Python 腳本，用於抓取台灣運彩的賽事資料和對應的盤口賠率，並整理出每場比賽的「不讓分」、「讓分」和「大小」盤口結論。

## 功能
1. 從指定的 API 獲取最新的比賽時程 (`BetSchedule.json`)。
2. 根據賽事資訊，動態查找並獲取對應的賽前盤口賠率 (`*-Games.zh.json`)。
3. 將賽事與盤口進行匹配。
4. 為每場比賽生成易於閱讀的盤口分析結論。
5. 將分析結果輸出為 Markdown 檔案。

## 使用方法
1. 確保已安裝 Python 及 `requests` 函式庫。
   ```bash
   pip install requests
   ```
2. 執行主腳本：
   ```bash
   python analyze_odds.py
   ```
3. 腳本執行完畢後，會在同目錄下生成一個 `odds_analysis_report.md` 檔案，其中包含所有比賽的盤口結論。

## 注意事項
- 本工具僅供學習和研究目的。
- API 網址可能會變更，如果腳本失效，請檢查 API 的可用性。
- 請遵守台灣運彩的相關使用規定。
